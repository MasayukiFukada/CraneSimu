# AI Agent Guide for Crane Simulator

このプロジェクトは、Three.js と cannon-es を使用したクレーンゲーム（橋渡し設定）の物理シミュレーターです。

## 🚀 プロジェクトの目的
ブラウザ上でリアルな「橋渡し」の挙動を再現し、物理演算（摩擦、重心、衝突）に基づいたシミュレーションを行うことを目的としています。

## 🛠 技術スタック
- **ビルドツール**: Vite
- **ランタイム管理**: mise (Node.js v22+)
- **3Dエンジン**: Three.js
- **物理エンジン**: cannon-es
- **言語**: TypeScript

## 📂 主要ファイル構造
- `src/main.ts`: シミュレーターのコアロジック（物理ワールド、レンダリング、操作）
- `src/style.css`: UIボタンのレイアウト
- `index.html`: UIコンテナとCanvas

## ⚙️ 物理演算の調整ポイント (重要)
シミュレーションのリアリティを高めるために、以下の値を `src/main.ts` で調整してください。

1.  **摩擦係数 (Friction)**: `CANNON.ContactMaterial` を作成し、橋の棒と景品、景品とクレーンの間の摩擦を設定します。
2.  **景品の重心**: `CANNON.Body` の形状オフセットを調整することで、重心の偏りを再現できます。
3.  **クレーンの強さ**: `isDropping` シーケンス内の上昇・下降速度や、クレーンが景品に与える物理的な力を調整します。

## 📝 開発時のガイドライン
- **mise の使用**: 依存関係のインストールや開発サーバの起動には `mise exec -- npm ...` を使用してください。
- **物理とメッシュの同期**: `animate()` ループ内で、常に `cannon-es` の `Body` の位置と回転を `Three.js` の `Mesh` に同期させてください。
- **UIの追加**: 操作ボタンを増やす場合は、`index.html` にボタンを追加し、`src/main.ts` の `setupUI()` でイベントリスナーを登録してください。
